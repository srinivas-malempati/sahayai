## 📱 What is SahayAI?

SahayAI is a **free, single-page web app** designed to help Telugu-speaking people — especially **elderly citizens with low digital literacy** — easily access information about government services, hospitals, and temples in Andhra Pradesh.

The app answers the most common question every AP citizen faces:

> *"నాకు ఏ పత్రాలు కావాలి? ఏ కార్యాలయానికి వెళ్ళాలి? ఎంత రుసుము చెల్లించాలి?"*  
> *"What documents do I need? Which office? How much does it cost?"*

---

## ✨ Key Features

- 🌐 **Telugu + English** — One tap to switch all content
- 📍 **GPS-based location detection** — Find nearest MeeSeva offices and hospitals
- 🎤 **Voice search** — Speak in Telugu or English to find services
- 📋 **75 government services** — Complete guide with documents, steps, fees
- 🏥 **Hospital finder** — GPS-sorted nearest hospitals with call + maps buttons
- 🛕 **Temple guide** — TTD darshan booking, sevas, VIP pass, accommodation
- 🚌 **Travel guide** — Bus + train routes from 5 cities to 3 major temples
- 📱 **Mobile-first** — Designed for iPhone/Android, works in any browser
- ⚡ **No login, no signup, no app install** — Just open and use
- 🔍 **Alphabetical service listing** — Easy for elderly to scan and find

## 🛠️ Tech Stack

```
Single HTML file — no framework, no build step, no dependencies
├── Pure HTML5 + CSS3 + Vanilla JavaScript
├── Google Fonts (Baloo 2, Noto Sans Telugu, Poppins)
├── Web Speech API (voice search)
├── Geolocation API (GPS-based nearest office/hospital)
├── Nominatim API (reverse geocoding — city detection)
└── Google Maps (deep links for directions)
```

**Why single file?**
- Zero setup — just open `index.html` in any browser
- Works offline after first load
- Easy to deploy anywhere (GitHub Pages, Vercel, Netlify)
- No Node.js, npm, or build tools required

## 🚀 Deployment

### Option 1 — GitHub Pages (Free)

```bash
# 1. Fork or clone this repo
git clone https://github.com/msrinivas6/sahayai.git

# 2. Go to repo Settings → Pages
# 3. Source: Deploy from branch → main → / (root)
# 4. Your app is live at: https://msrinivas6.github.io/sahayai
```

### Option 2 — Vercel (Recommended)

```bash
# Install Vercel CLI
npm install -g vercel

# Deploy (from repo folder)
vercel --prod --force
```

Or connect your GitHub repo to [vercel.com](https://vercel.com) for automatic deployments on every push.

### Option 3 — Local

```bash
# Just open the file directly
open index.html   # macOS
start index.html  # Windows
```

## 🎯 Design Philosophy

### Grandfather First
Every design decision is made for a **70-year-old in Vijayawada** who:
- Has a basic Android/iPhone
- Can read Telugu but limited English
- Has never used a government website
- Gets confused by too many menus

**Principles:**
- Large text (17px base, Baloo 2 font for headings)
- Minimum 56px touch targets on all buttons
- Maximum 2 taps to reach any information
- Warm Indian colors (saffron/green/cream) — NOT dark mode
- Telugu first, English second
- No login, no forms, no complexity

### Information Accuracy
Every service has been verified with:
- Official government websites (ap.meeseva.gov.in, meebhoomi.ap.gov.in, etc.)
- Phone numbers tested before adding
- Fees and documents verified as of 2025
- Warnings added for common mistakes people make
